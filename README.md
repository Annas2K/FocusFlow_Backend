#  FocusFlow Project - Week 9
## Cách chạy dự án
1. Đảm bảo máy đã cài JDK 17.
2. Mở Terminal tại thư mục `target`.
3. Chạy lệnh: `java -jar FocusFlow-0.0.1-SNAPSHOT.jar`

## Tài liệu API (Swagger)
Sau khi chạy app, truy cập: http://localhost:8080/swagger-ui/index.html, hiện tại vẫn chưa có giao diện swagger và sẽ fix trong tương lai
